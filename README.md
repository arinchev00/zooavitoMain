# ZooAvito

**ZooAvito** — это веб-приложение для размещения объявлений о животных, зоотоварах и услугах, связанных с питомцами. Платформа позволяет пользователям создавать объявления, просматривать их, управлять профилем, а администраторам — управлять категориями и подкатегориями.

## Основные возможности

### Для всех пользователей
- **Просмотр объявлений** — лента объявлений с фильтрацией по категориям и подкатегориям
- **Поиск объявлений** — поиск по заголовку и описанию
- **Фильтрация объявлений** — фильтрация по категориям, подкатегориям, цене от и до
- **Детальная страница объявления** — просмотр полной информации, фото, автора объявления, комментарий
- **Регистрация и авторизация** — создание аккаунта и вход в систему
- **Просмотр категорий** — древовидная структура категорий и подкатегорий

### Для авторизованных пользователей
- **Создание объявлений** — добавление объявлений с заголовком, описанием, ценой, категорией и фото
- **Редактирование объявлений** — изменение данных и управление фотографиями (только своего)
- **Удаление объявлений** — удаление объявления (только своего)
- **Создание комментария** — добавление комментария к любому обьявлению
- **Редактирование комментария** — изменение текста комментария (только своего)
- **Удаление комментария** — удаление комментария (только своего)
- **Личный кабинет** — просмотр и редактирование профиля (Полное имя, телефон, пароль)
- **Мои объявления** — управление своими объявлениями
- **Загрузка фотографий** — до 10 фото на объявление с возможностью выбора главного

### Для администратора
- **Управление категориями** — создание, редактирование, удаление категорий
- **Управление подкатегориями** — создание, редактирование, удаление подкатегорий
- **Управление порядком категорий** — изменение порядка отображения категорий на главной
- **Панель администратора** — централизованное управление всем контентом
- **Изменение роли пользователям** — возможность назначить любого пользователя Администратором
- **Блокировка пользователей** — возможность блокировки нарушителей
- **Создание объявлений** — добавление объявлений с заголовком, описанием, ценой, категорией и фото
- **Редактирование объявлений** — изменение данных и управление фотографиями (для всех объявлений)
- **Удаление объявлений** — удаление объявления (для всех объявлений)
- **Создание комментария** — добавление комментария к любому обьявлению
- **Редактирование комментария** — изменение текста комментария (только своего)
- **Удаление комментария** — удаление комментария (для всех комментариев)

## Технологический стек

### Frontend
| Технология | Назначение |
|------------|------------|
| **React 19** | Библиотека для построения пользовательского интерфейса |
| **React Router DOM** | Маршрутизация |
| **Axios** | HTTP-запросы |
| **Bootstrap 4** | CSS-фреймворк |
| **Font Awesome** | Иконки |
| **dnd-kit** | Drag-and-drop для управления порядком категорий |
| **jwt-decode** | Декодирование JWT токенов |

### Backend
| Технология | Назначение |
|------------|------------|
| **Spring Boot 3** | Основной фреймворк |
| **Spring Security** | Аутентификация и авторизация |
| **JWT** | Токены для авторизации |
| **Spring Data JPA** | Работа с базой данных |
| **PostgreSQL** | База данных |
| **Flyway** | Миграции базы данных |
| **Hibernate** | ORM |
| **Maven** | Сборка проекта |

### Инфраструктура
| Технология | Назначение |
|------------|------------|
| **Docker** | Контейнеризация приложения |
| **Nginx** | Веб-сервер для статики и проксирования |
| **Docker Compose** | Оркестрация контейнеров |

## Установка и запуск

### Требования
- Docker и Docker Compose
- Java 21 (для локальной сборки бэкенда)
- Node.js 20 (для локальной сборки фронтенда)

### Запуск через Docker (рекомендуемый способ)

```bash
# Клонировать репозиторий
git clone https://github.com/arinchev00/zooavitoMain.git
cd zooavitoMain

# Создать файл с переменными окружения
cp .env.example .env

# Отредактировать .env, добавив соответствующие переменные из .env.example

# Собрать и запустить контейнеры
docker-compose up -d --build

# Проверить статус
docker-compose ps
```

### Требования к загрузке изображений

| Параметр | Значение |
|------------|------------|
| **Максимальный размер одного файла** | 10 МБ |
| **Максимальный общий размер запроса** | 50 МБ |
| **Максимальное количество фото** | 10 шт. |
| **Поддерживаемые форматы** | JPG, JPEG, PNG, WEBP, GIF |
